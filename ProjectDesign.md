# Employee Management System

## Overview
A console-based Java project to manage employee details with OOP and SQL.

## Classes
- Employee: id, name, salary
- Manager: extends Employee, adds bonus

## SQL Tables
- Employee table: id, name, salary
- Department table: id, name

## Features
- Add / Update / Delete employees
- Display employee & manager details
- SQL queries: SELECT, JOIN, GROUP BY
- Exception handling in Java
